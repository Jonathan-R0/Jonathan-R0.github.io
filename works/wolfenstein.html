Wolfenstein Fiuba

This game was developed as a University project. The game has almost entirely been developed in C++ and documented with Doxygen. The game offers a local multiplayer expercience, with a Lua developed AI. The Lua module can be changed by the users to create unique enemies.

Product

The Raycaster engine was developed using SDL.
